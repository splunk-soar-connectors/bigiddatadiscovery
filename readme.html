The BigID Data Discovery App for Splunk SOAR provides functionality allowing business users to take action on their data, regardless of the type or location. With actions for searching BigID's data catalog, retrieving PII findings, executing a scan for given data sources and retrieving those scan results, as well as retrieving BigID audit logs, business users will have an invaluable tool for building robust playbooks to action and protect sensitive data.
